Projet : Clone Facebook (version simplifiée)
Réalisé par : Hamza Sayari
Matière : Développement Web / JavaScript
Date : 29 novembre 2025
Objectif du projet
Créer une petite application web qui ressemble à un profil Facebook en utilisant uniquement HTML, CSS et JavaScript vanilla (sans framework ni bibliothèque externe).
L’objectif était de mettre en pratique la manipulation du DOM, les événements, le chargement de données JSON et la création dynamique de contenu.
Fonctionnalités implémentées

Affichage d’un profil complet (photo, nom, nombre d’abonnés)
Trois onglets : Posts, À propos, Amis
Chargement des posts, commentaires et amis depuis un fichier data.json
Système de Like fonctionnel avec compteur et changement de style
Affichage/masquage des commentaires
Section À propos avec sous-menus (Overview, Travail & études, Ville vécue)
Possibilité de modifier son nom dans le profil
Création de nouveaux posts en temps réel (zone « Quoi de neuf ? ») → les nouveaux posts s’affichent immédiatement en haut
Recherche en temps réel dans la liste d’amis
Système de notifications avec cloche et compteur de non-lus
Design proche de l’interface réelle de Facebook (couleurs, ombres, boutons arrondis, etc.)

Structure des fichiers

index.html → structure principale de la page
style.css → tout le style (responsive partiel inclus)
script.js → toute la logique JavaScript
data.json → données des utilisateurs, posts et notifications
